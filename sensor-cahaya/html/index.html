Ikuti langkah berikut untuk mencoba sensor cahaya daeri kami:


  1. Untuk Menjalankan Sensor Cahaya Ini Harus Menggunakan Extentions VsCode LIVE SERVER

  2. Klik tombol Mulai lalu izinkan akses kamera.

  3. Mainkan kamera dengan tangan atau menggunakan cahaya, agar bisa terlihat sensornya

  4. Jika lampu Kuning yang Menyala = cahaya area sekitar gelap/kurang pencahayaan,
    Jika lampu Abu abu yang menyala = keadaan cahaya sekitar cukup (terang).

  5. Jika sudah selesai mendemokan aplikasi nya tekan tombol Berhenti.
